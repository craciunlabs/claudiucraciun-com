# claudiucraciun.com

Personal site for Claudiu Craciun — business strategist, tech professional, and plant medicine practitioner.

## About

Two worlds. One path.

Twenty years in business and technology. Eight years working with traditional Amazonian plant medicine. This site lives at the intersection of both.

**Live site:** [claudiucraciun.com](https://claudiucraciun.com)

| File | Purpose |

## Design System

- **Background:** `#0D0C0A` (near-black)
- **Primary gold:** `#C9A84C` / hover `#E8C96A`
- **Typography:** Cormorant Garant (display serif) + Switzer (body sans)
- **Aesthetic:** Luxury editorial — The Row, Bottega Veneta. No spiritual clichés.

## Deployment

Static HTML/CSS/JS — no build step required. Deploy directly to Vercel, Netlify, or any static host.

```bash
# Vercel CLI
vercel --prod
```
